# planaria_combined

"Verification of quantum biological predictions for weak magnetic field effects on superoxide in planarians" by Rishabh, Jana Vučković, Hadi Zadeh-Haghighi, Wendy S. Beane, and Christoph Simon (2026).

This repository contains:
1. Mathematica notebook which was used to generate theoretical plots for the manuscript and the supporting information. Mathematica Version 13.1 (2023) was used to write the code. For computations done using the Advanced Research Computing (ARC) cluster at the University of Calgary, Mathematica Version 11.0 was used.

2. Sample python code for parameter search described in Section 2.3. Python 3 was used. Search was conducted on Google Colab platform.
